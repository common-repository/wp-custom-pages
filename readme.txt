Plugin for creating pages with exact structure.

== Description ==
This plugin is for adding pages in tree structure with custom strucure. Add ability sort pages by several automatic generated taxonomies. Or you can generate one page from all subpages.

= Required: =
* This plugin requires Nextgen Gallery plugin!!!

= Recommended: =
* Plugin CMS Tree Page View is highly recommended.

= What this plugin can do: =
* Create groups with custom page structure
* Add taxonomies (from selects, checkboxes, page group and parent pages titles)
* Add page content on the top
* Add subpages tree on the bottom
* Add same level pages list on the top (ohter 10 pages)
* Add page path on the top
* Generate one superpage from selected page and all subpages with content. Save as post and html.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to use this plugin =
* Create at least one group and you can add articles with structure of the created group.

= Taxonomies =
= Default taxonomies =
* Typ: name of the group
* Filter: all used taxonomies + all parent pages and page title

= Custom taxonomies =
* Taxonomies are created from Selects and Checkboxes (your_url.com/select_name/select_value)

= Groups =
* By the group is meant the structure of Article.
* You can choose from five different types of inputs (Text, Textarea, Select, Checkbox, Picture).
* Title and Custom text (classic TinyMCE WYSIWYG editor) are required rows.

= Select =
* From select are automatically created taxonomies.
* You can choose from values used before or add new one.

= Checkbox =
* Looks like Select but you can choose from only two options.

= Picture =
* You can add Nextgen Gallery or Picture from Nextgengallery
